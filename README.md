# shrec19-shape-retrieval
## Requirements
 

```
Blender 2.79
Python 3.x
```
## Folder structure

```
	<output_data_folder>
	
	|- class_i
		|- obj
			|- 0001.png
			|- 0002.png
			|- ...
	|- ...
		
```

## Usage
```
blender -b -P <path to script>
```
*Example*
```
blender -b -P '/home/ken/scripts blender/run.py'
```
